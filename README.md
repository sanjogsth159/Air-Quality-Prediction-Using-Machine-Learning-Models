# Air-Quality-Prediction-Using-Machine-Learning-Models
This project analyzes the Air Quality UCI dataset using machine learning regression techniques including Linear Regression, Random Forest Regressor, and KNN Regressor. The study focuses on data preprocessing, exploratory data analysis, model training, evaluation, and performance comparison to predict air pollutant concentrations.


#Citation
Vito, S. (2008). Air Quality [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C59K5F.
